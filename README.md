### Smart Stop Watch 

## Description

Smart Stop Watch is a menu-driven C program designed to help users manage daily health-related activities such as tracking steps, monitoring heart rate, setting hydration reminders, and managing daily routines. The project demonstrates the use of core C programming concepts in a practical way.

## Features

Track daily steps (in kilometers)

Check heart rate status

Hydration reminder based on time

Set and complete daily routines

Reward points system for motivation

Simple menu-driven interface

## Technologies Used

Programming Language: C

Compiler: GCC

Editor: Visual Studio Code

## How to Run

Compile the program:

gcc main.c -o main


## Run the program:

./main        # macOS / Linux
main.exe     # Windows

## Project Files

main.c – Source code of the project

output.txt – Sample output of the program

## Result

This project helps beginners understand functions, pointers, loops, conditionals, and basic input/output in C through a real-life inspired application.
